# ディープリサーチでリストを作成するときの依頼文（コピー用）

ディープリサーチやAIに「鶴岡市の子ども連れ向けスポット」を調べてもらうとき、**下の依頼文をそのままコピーして使ってください**。  
回答をそのままサイトのデータ（JSON）に貼り付けできる形式で出してもらえます。

---

## 依頼文（このブロックをコピーして使う）

```
山形県鶴岡市で、子ども連れ（乳幼児〜小学生）が行きやすいお店・施設・スポットを調査し、一覧にまとめてください。

【出力形式】
必ず次のJSON形式で出力してください。ほかの説明文は「remarks」にまとめ、メインの回答は「places」配列のみでお願いします。

{
  "places": [
    {
      "id": "英数字のユニークID（例: place-01）",
      "name": "施設・お店の正式名称",
      "categories": ["eat", "cafe"],
      "primaryCategory": "eat",
      "address": "住所（できるだけ正式な表記）",
      "phone": "電話番号（わかれば）",
      "hours": "営業・開館時間（例: 9:00～17:00）",
      "closed": "定休日（例: 月曜日、年末年始）",
      "description": "子連れ向けの特徴を2〜3文で",
      "details": {}
    }
  ]
}

【カテゴリ（複数可）】
- eat：ランチ・ごはん（飲食店で食事）
- cafe：カフェ・スイーツ
- babyStation：赤ちゃんの駅（授乳・オムツ替え）
- playIndoor：室内で遊ぶ（児童館、キッズ施設等）
- playOutdoor：公園・外あそび
- aquarium：水族館・動物
- onsen：温泉・宿泊
- beauty：美容室・サロン
- library：図書館・読み聞かせ
- craft：ものづくり・体験
- museum：博物館・歴史
- nature：自然・ハイキング

各スポットは "categories": ["eat", "cafe"] のように複数指定可。"primaryCategory": "eat" でメイン1つを指定。

【details の内容（primaryCategory に応じて）】
- eat / cafe: kozakai, koshitsu, junyushitsu, babyChair, kidsMenu, omutsu, parking
- playIndoor: indoor, ageRange, junyushitsu, omutsu, restSpace, parking
- playOutdoor / nature: stroller, toilet, omutsu, parking
- aquarium: indoor, ageRange, junyushitsu, omutsu, parking
- onsen: familyBath, babyAmenity, parking
- beauty: kidsSpace, private, parking
- library: ageRange, event, parking
- craft / museum: ageRange, parking
- babyStation: junyushitsu, omutsu, parking

わからない項目は省略するか空文字 "" にしてください。公式サイト・自治体の子育てガイド・口コミ等を参照し、可能な範囲で正確な情報を入れてください。
```

---

## 使い方

1. 上記の依頼文をコピーし、ディープリサーチ（または使用しているAI）に貼り付けて実行する。
2. 返ってきた **「places」配列** をコピーする。
3. プロジェクトの **`data/places.json`** を開き、既存の `"places": [ ... ]` の `...` 部分を、新しいリストの内容と差し替える（または既存の後ろに追加する）。
4. サイト表示用にデータを埋め込んでいる場合は、**`app.js`** の先頭にある `PLACES_DATA` の `places` 部分も同様に更新する（またはローカルサーバーで `data/places.json` を読み込む運用にする）。

---

## 補足

- **id** は重複しないように、例として `manma-room`, `cafe-01`, `park-01` のような英数字で付けてください、と依頼に追記してもよいです。
- 調査結果を **CSV** で欲しい場合は、「上記の各項目を列にしたCSVで出力し、1行目はヘッダー（id, name, category, address, ...）にしてください」と追加すると、後からJSONに変換しやすくなります。
